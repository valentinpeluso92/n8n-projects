Llama a esta tool para modificar la fecha o la hora de un turno existente para el paciente actual. Un paciente solo puede modificar sus propios turnos.
Los inputs deben ser:
- id_turno: ID del turno a modificar
- dni: DNI sin puntos ni guiones en formato string
- nueva_fecha: Nueva fecha en formato (DD/MM/YYYY)
- nueva_hora: Nueva hora en formato (HH:MM)
