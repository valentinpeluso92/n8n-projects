
Llama a esta tool cuando necesites derivar un caso a la secretaria humana cuando el agente no puede resolver la consulta o hay una situación que requiere intervención humana.
Los inputs deben ser:
- nombre_completo: Nombre completo del paciente
- telefono: Teléfono de contacto del paciente
- motivo: Motivo de la derivación: "urgencia", "receta", "presupuesto", "obra_social", "error_tecnico", "modificacion_urgente", "consulta_compleja", "otro"
- dni: DNI sin puntos ni guiones en formato string
- observaciones: Detalles adicionales
- turno_relacionado: ID del turno relacionado
- prioridad: Nivel de prioridad: "alta", "media", "baja"
